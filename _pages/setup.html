Setting up Google Play Music Desktop Player

For this skin to function properly, you must first install Google Play Music Desktop Player. After installation, you then need to navigate to the applications Desktop Settings, where you must then ensure that "Enable Playback API" and "Enable JSON API" are selected, as featured in the image below.

Authorizing the GPMDP Rainmeter plugin

While the inclusion of media controls into PlayMusic has been requested since the initial release, remote functionalities have been impossible until the recent release of GPMDP-Plugin by tjhrulz, which cleverly uses the GPMDP web-socket to allow remote control of the application. Though, in order for the plugin to work properly, the user must provide an four-digit authorization code obtained via the GPMDP application. To make this process as easy as straightforward as possible, a supplementary authorization skin has been included in PlayMusic, featuring a prompt that provides confirmation of the authorization state.

Troubleshooting

If you run into any issues throughout the setup proces, I would first suggest refreshing the skin and/or relaunching the GPMDP application. Though, if the problem persists, feel free to open an issue here.
Also, if you have an issue during the authentication process of the GPMDP plugin, I would suggest that you visit the Troubleshooting page of the GPMDP-Plugin wiki.
